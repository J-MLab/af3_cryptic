# **Analysis code for the paper "The Influence of Ligands on AlphaFold3 Prediction of Cryptic Pockets" by Lazou et al.**

The following scripts were used for the analysis and visualization in the manuscript.

## Initialize the environment
```bash
uv sync
```

## Run a script with the environment
```bash
uv run python python_script.py
```
